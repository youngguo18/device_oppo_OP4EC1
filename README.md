Device configuration for OPPO Reno6 Pro+ 5G
=========================================

The OPPO Reno6 Pro+ 5G (codenamed _"OP4EC1"_) is a high-end smartphone from OPPO.

It was announced & released in March 2020.

## Device specifications

Basic   | Spec Sheet
-------:|:-------------------------
SoC     | Qualcomm Snapdragon 870 SM8250
CPU     | Octa-Core (1 x 2.84 GHz Kryo 585 & 3 x 2.42 GHz Kryo 585 & 4x1.8 GHz Kryo 585)
GPU     | Adreno 650
Memory  | 12 GB RAM
Shipped Android Version | 11.0 with ColorOS 11.3
Storage | 256GB
Battery | Non-removable Li-Po 4500 mAh battery
Display | 1080 x 2400 pixels, 20:9 ratio, 6.62 inches AMOLED (~398 ppi density) |
Rear Camera  | 50MP (wide) / 13MP (telephoto) / 16MP (ultrawide) / 2MP (macro)
Front Camera | 32MP (wide)
